# Percentage types

_Percentage and basis-point representations_

**Schema**: `core/percentage.schema.json`

## Percentage

Decimal percentage (0.08 = 8%). Soft max of 1 (100%); some values may exceed.

**Type**: `number`
**Constraints**: min: 0, max: 1

Decimal percentage (0.08 = 8%). Soft max of 1 (100%); some values may exceed.

## BasisPoints

Rate in basis points (100 = 1%)

**Type**: `integer`
**Constraints**: min: 0

Rate in basis points (100 = 1%)
